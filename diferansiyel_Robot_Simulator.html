Giriş Parametreleri

Bu modelde gövde eksen takımında yanal hız vy = 0 kabul edilir. Robot yanal kaymaz. Ancak dünya koordinat sisteminde ẏ sıfır olmayabilir.

Kinematik Formüller

r = D / 2
Vl = 2πr · RPMsol / 60
Vr = 2πr · RPMsağ / 60
Vx = (Vr + Vl) / 2
Vy = 0     (gövde eksen takımında)
θ̇ = (Vr - Vl) / L
ẋ = Vx cos(θ)
ẏ = Vx sin(θ)
θ(t+Δt) = θ(t) + θ̇Δt
